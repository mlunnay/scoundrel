Class DiscSectorZone

The DiscSectorZone zone defines a section of a Disc zone. The disc on which it's based have a hole in the middle, like a doughnut.

constructor

  • new DiscSectorZone(center?: Vector2, outerRadius?: number, innerRadius?: number, minAngle?: number, maxAngle?: number): DiscSectorZone
    • Defined in flec/zones/discsectorzone.ts:21

    The constructor defines a DiscSectorZone zone.

    Parameters

    • Default value center: Vector2 = new Vector2(0,0)

      The centre of the disc.

    • Default value outerRadius: number = 0

      The radius of the outer edge of the disc.

    • Default value innerRadius: number = 0

      If set, this defines the radius of the inner edge of the disc. Points closer to the center than this inner radius are excluded from the zone. If this parameter is not set then all points inside the outer radius are included in the zone.

    • Default value minAngle: number = 0

      The minimum angle, in radians, for points to be included in the zone. An angle of zero is horizontal and to the right. Positive angles are in a clockwise direction (towards the graphical y axis). Angles are converted to a value between 0 and two times PI.

    • Default value maxAngle: number = 0

      The maximum angle, in radians, for points to be included in the zone. An angle of zero is horizontal and to the right. Positive angles are in a clockwise direction (towards the graphical y axis). Angles are converted to a value between 0 and two times PI.

    Returns DiscSectorZone

collideParticle

  • collideParticle(particle: Particle, bounce?: number): boolean

  • Implementation of Zone.collideParticle

    • Defined in flec/zones/discsectorzone.ts:249

    Manages collisions between a particle and the zone. The particle will collide with the edges of the disc sector defined for this zone, from inside or outside the disc. In the interests of speed, these collisions do not take account of the collisionRadius of the particle.

    Parameters

    • particle: Particle

      The particle to be tested for collision with the zone.

    • Default value bounce: number = 1

      The coefficient of restitution for the collision.

    Returns boolean

    Whether a collision occured.
